# threejsLearn
Three.js Learning Workspace

This is a monorepo workspace using pnpm for learning and experimenting with Three.js.

## Features
- pnpm workspace configuration
- Scalable monorepo structure
- Ready for multiple Three.js projects/examples

## Workspace Structure
```
├── packages/        # Your individual projects/packages go here
├── package.json    # Workspace configuration
└── README.md       # This file
```

## Getting Started
1. Install dependencies: `pnpm install`
2. Add your projects to the `packages/` directory
3. Run scripts from root or individual packages
